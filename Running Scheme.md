### Key principles (read this once — it makes the plan work)
- **Easy pace wins**: most runs should feel *almost too easy*. You should be able to **talk in full sentences**.
- **Run-walk is smart** (especially at 90 kg): it lowers impact and helps consistency. You can still run a great 15 km with run-walk.
- **Pain rule**:
  - Normal: mild muscle soreness, stiffness that improves as you warm up.
  - Not normal: sharp pain, limping, pain that worsens while running, or pain that lasts >48h → rest and adjust.
- **Progression rule**: if a week feels too hard, **repeat the week** instead of pushing forward.
- **Surface**: prefer flat, forgiving paths (park, gravel) early on.
- **Shoes**: good running shoes (ideally fitted) + socks that prevent blisters.

## How to run each workout

### Effort scale (simple)
- **Easy (E)**: comfortable, can talk; RPE **3–4/10**
- **Steady (S)**: “comfortably challenging,” short sentences; RPE **6/10**
- **Tempo (T)**: controlled hard, 20–30 min sustainable; RPE **7/10**
- **Intervals (I)**: short, strong effort but controlled; RPE **8/10** (never sprinting)

### Run-walk options (use whenever you like)
Pick the ratio that keeps you **easy**:
- 1 min run / 1 min walk
- 2/1
- 3/1
- 4/1
- 5/1

### Warm-up & cool-down (every run)
- **Warm-up (5–8 min):** brisk walk + very easy jog
- **Cool-down (5 min):** walk
- Optional: 5 minutes mobility (ankles, calves, hips)

## Weekly structure (recommended)
You’ll run **3x/week** (sometimes 4 if recovery is excellent) + **2x strength**.

**Template week (move days as needed):**
- Day 1: Rest / walking
- Day 2: Run A (easy)

- Day 4: Run B (easy + small “quality” later)

- Day 6: Run C (easy) *(optional, if you’re recovering well)*

If you only want **3 runs/week**, skip **Run C** and keep walking/strength.

## Strength training (2×/week, 20–35 min)
**Goal:** stronger calves/hips/core → less injury risk + better running economy.

Pick **5–7 exercises**, do **2–3 sets** of **8–12 reps** (controlled):
- Box squat (sit to a bench/chair)
- Romanian deadlift (light) or hip hinge
- Step-ups (low step)
- Glute bridge / hip thrust
- Calf raises (slow)
- Side plank
- Dead bug (core)
- Band walks (hip stability)

Keep it “easy-medium” at first. Consistency beats intensity.

---

## Weight loss (simple, sustainable)
**Target:** ~**0.25–0.5 kg per week** (supports recovery while losing fat)

- **Calorie deficit:** roughly **300–500 kcal/day** below maintenance.
- **Protein:** aim **120–160 g/day** (satiety + muscle retention).
- **Daily steps:** build toward **7,000–10,000/day**.
- **Sleep:** aim **7–8 hours** (hunger control + recovery).
- **Limit liquid calories:** alcohol/soft drinks are usually the easiest win.

## Race-day pacing (so 15 km feels “easy”)
- **Km 1–3:** very easy (slower than you think)
- **Km 4–12:** steady, controlled
- **Last 3 km:** only push if you still feel good

Run-walk on race day is totally acceptable:
- Example: **8–10 min run / 1 min walk** from the start can feel amazing at 15 km.
